Detailed Description

GETRI_NPVT_OUTOFPLACE computes the inverse \(C = A^{-1}\) of a general n-by-n matrix A without partial pivoting.

The inverse is computed by solving the linear system

\[ AC = I \] 

where I is the identity matrix, and A is factorized as \form#146 as given by \ref rocsolver_sgetrf_npvt "GETRF_NPVT".
