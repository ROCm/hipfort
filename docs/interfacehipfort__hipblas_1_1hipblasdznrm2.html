Detailed Description

!

BLAS Level 1 API

 nrm2 computes the euclidean norm of a real or complex vector

           result := sqrt( x'x ) for real vectors
           result := sqrt( xHx ) for complex vectors
Parameters
[in]handle[hipblasHandle_t] handle to the hipblas library context queue.
[in]n[int] the number of elements in x.
[in]xdevice pointer storing vector x.
[in]incx[int] specifies the increment for the elements of y.
[in,out]resultdevice pointer or host pointer to store the nrm2 product. return is 0.0 if n, incx<=0.
