Detailed Description

!

GETRI_NPVT_BATCHED inverts a batch of general n-by-n matrices using the LU factorization computed by GETRF_NPVT_BATCHED.

 The inverse of matrix \form#72 in the batch is computed by solving the linear system

\[ A_j^{-1} L_j = U_j^{-1} \] 

where \form#85 is the lower triangular factor of \form#72 with unit diagonal elements, and \form#106 is the
upper triangular factor.
Parameters
[in]handlerocblas_handle.
[in]nrocblas_int. n >= 0.
The number of rows and columns of all matrices A_j in the batch.
[in,out]Aarray of pointers to type. Each pointer points to an array on the GPU of dimension ldan.
On entry, the factors L_j and U_j of the factorization A = L_jU_j returned by GETRF_NPVT_BATCHED. On exit, the inverses of A_j if info[j] = 0; otherwise undefined.
[in]ldarocblas_int. lda >= n.
Specifies the leading dimension of matrices A_j.
[out]infopointer to rocblas_int. Array of batch_count integers on the GPU.
If info[j] = 0, successful exit for inversion of A_j. If info[j] = i > 0, U_j is singular. U_j[i,i] is the first zero pivot.
[in]batch_countrocblas_int. batch_count >= 0.
Number of matrices in the batch.
