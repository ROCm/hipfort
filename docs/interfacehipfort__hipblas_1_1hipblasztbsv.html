Detailed Description

!

BLAS Level 2 API

 tbsv solves

      Ax = b or ATx = b or AHx = b,

 where x and b are vectors and A is a banded triangular matrix.
Parameters
[in]handle[hipblasHandle_t] handle to the hipblas library context queue.
[in]uplo[hipblasFillMode_t] HIPBLAS_FILL_MODE_UPPER: A is an upper triangular matrix. HIPBLAS_FILL_MODE_LOWER: A is a lower triangular matrix.
[in]transA[hipblasOperation_t] HIPBLAS_OP_N: Solves Ax = b HIPBLAS_OP_T: Solves ATx = b HIPBLAS_OP_C: Solves AHx = b
[in]diag[hipblasDiagType_t] HIPBLAS_DIAG_UNIT: A is assumed to be unit triangular (i.e. the diagonal elements of A are not used in computations). HIPBLAS_DIAG_NON_UNIT: A is not assumed to be unit triangular.
[in]n[int] n specifies the number of rows of b. n >= 0.
[in]k[int] if(uplo == HIPBLAS_FILL_MODE_UPPER) k specifies the number of super-diagonals of A. if(uplo == HIPBLAS_FILL_MODE_LOWER) k specifies the number of sub-diagonals of A. k >= 0.
[in]Adevice pointer storing the matrix A in banded format.
[in]lda[int] specifies the leading dimension of A. lda >= (k + 1).
[in,out]xdevice pointer storing input vector b. Overwritten by the output vector x.
[in]incx[int] specifies the increment for the elements of x.
