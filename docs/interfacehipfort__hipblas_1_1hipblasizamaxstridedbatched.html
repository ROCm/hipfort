Detailed Description

BLAS Level 1 API.

amaxStridedBatched finds the first index of the element of maximum magnitude of each vector x_i in a batch, for i = 1, ..., batchCount.

Parameters
[in]handle[hipblasHandle_t] handle to the hipblas library context queue.
[in]n[int] number of elements in each vector x_i
[in]xdevice pointer to the first vector x_1.
[in]incx[int] specifies the increment for the elements of each x_i. incx must be > 0.
[in]stridex[hipblasStride] specifies the pointer increment between one x_i and the next x_(i + 1).
[in]batchCount[int] number of instances in the batch
[out]resultdevice or host pointer for storing contiguous batchCount results. return is 0 if n <= 0, incx<=0.
