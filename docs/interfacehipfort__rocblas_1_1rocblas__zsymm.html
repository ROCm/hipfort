Detailed Description

BLAS Level 3 API.

symm performs one of the matrix-matrix operations:

C := alpha*A*B + beta*C if side == rocblas_side_left, C := alpha*B*A + beta*C if side == rocblas_side_right,

where alpha and beta are scalars, B and C are m by n matrices, and A is a symmetric matrix stored as either upper or lower.

Parameters
[in]handle[rocblas_handle] handle to the rocblas library context queue.
[in]side[rocblas_side] rocblas_side_left: C := alpha*A*B + beta*C rocblas_side_right: C := alpha*B*A + beta*C
[in]uplo[rocblas_fill] rocblas_fill_upper: A is an upper triangular matrix rocblas_fill_lower: A is a lower triangular matrix
[in]m[rocblas_int] m specifies the number of rows of B and C. m >= 0.
[in]n[rocblas_int] n specifies the number of columns of B and C. n >= 0.
[in]alphaalpha specifies the scalar alpha. When alpha is zero then A and B are not referenced.
[in]Apointer storing matrix A on the GPU. A is m by m if side == rocblas_side_left A is n by n if side == rocblas_side_right only the upper/lower triangular part is accessed.
[in]lda[rocblas_int] lda specifies the first dimension of A. if side = rocblas_side_left, lda >= max( 1, m ), otherwise lda >= max( 1, n ).
[in]Bpointer storing matrix B on the GPU. Matrix dimension is m by n
[in]ldb[rocblas_int] ldb specifies the first dimension of B. ldb >= max( 1, m )
[in]betabeta specifies the scalar beta. When beta is zero then C need not be set before entry.
[in]Cpointer storing matrix C on the GPU. Matrix dimension is m by n
[in]ldc[rocblas_int] ldc specifies the first dimension of C. ldc >= max( 1, m )
