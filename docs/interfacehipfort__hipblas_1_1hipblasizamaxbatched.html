Detailed Description

!

BLAS Level 1 API

  amaxBatched finds the first index of the element of maximum magnitude of each vector x_i in a batch, for i = 1, ..., batchCount.
Parameters
[in]handle[hipblasHandle_t] handle to the hipblas library context queue.
[in]n[int] number of elements in each vector x_i
[in]xdevice array of device pointers storing each vector x_i.
[in]incx[int] specifies the increment for the elements of each x_i. incx must be > 0.
[in]batchCount[int] number of instances in the batch, must be > 0.
[out]resultdevice or host array of pointers of batchCount size for results. return is 0 if n, incx<=0.
