Detailed Description

!

GELQ2 computes a LQ factorization of a general m-by-n matrix A.

 (This is the unblocked version of the algorithm).

 The factorization has the form

\[ A = \left[\begin{array}{cc} L & 0 \end{array}\right] Q \] 

where L is lower triangular (lower trapezoidal if m > n), and Q is
a n-by-n orthogonalunitary matrix represented as the product of Householder matrices

\[ Q = H_k'H_{k-1}' \cdots H_1', \quad \text{with} \: k = \text{min}(m,n). \] 

Each Householder matrix \form#8 is given by

\[ H_i = I - \text{ipiv}[i] \cdot v_i' v_i \] 

where the first i-1 elements of the Householder vector \form#13 are zero, and \form#23.
